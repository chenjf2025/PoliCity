# 指标数据导入说明

## 一、导入文件格式要求

导入功能支持 **Excel 文件 (.xlsx)** 格式。

### 1.1 列格式

| 列名 | 说明 | 示例 |
|------|------|------|
| region_code | 地区代码 | default |
| region_name | 地区名称 | 测试城市 |
| E01 | 人均地区生产总值 | 12.50 |
| E02 | 地均GDP产出 | 1.85 |
| E03 | 第三产业占比 | 58.5 |
| ... | 其他指标列 | ... |

### 1.2 指标列说明

所有43个指标均可以作为列存在，每列的列名使用指标编码：
- **E01-E10**: 经济活力指标
- **C01-C08**: 文化繁荣指标
- **H01-H08**: 人力资源指标
- **U01-U08**: 城乡融合指标
- **G01-G09**: 城市治理指标

## 三、导入步骤

### 3.1 前端导入

1. 访问 http://192.168.71.127:3001
2. 进入 **指标管理** 页面
3. 点击 **导入数据** 按钮
4. 选择年份（默认2024）
5. 选择 Excel 文件
6. 点击确认导入

### 3.2 API导入

```bash
# 导入2024年数据
curl -X POST "http://192.168.71.127:8000/api/v1/data/raw/import?report_year=2024" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@年度数据导入模板_2024.xlsx"
```

### 3.3 单条数据录入

```bash
curl -X POST "http://192.168.71.127:8000/api/v1/data/raw" \
  -H "Content-Type: application/json" \
  -d '{
    "region_code": "default",
    "region_name": "测试城市",
    "indicator_code": "E01",
    "report_year": 2024,
    "raw_value": 12.50
  }'
```

## 四、导入后操作

### 4.1 触发标准化计算

导入原始数据后，需要执行标准化计算才能查看评分：

```bash
curl -X POST "http://192.168.71.127:8000/api/v1/data/normalize" \
  -H "Content-Type: application/json" \
  -d '{"report_year": 2024}'
```

### 4.2 验证导入结果

```bash
# 查看原始数据
curl "http://192.168.71.127:8000/api/v1/data/raw?region_code=default&report_year=2024&limit=10"

# 查看标准化得分
curl "http://192.168.71.127:8000/api/v1/data/standard-score?region_code=default&report_year=2024"
```

## 五、注意事项

1. **Excel格式**: 仅支持 .xlsx 格式，不支持 .xls 或 .csv
2. **指标列**: 指标列必须使用正确的编码（如 E01、C01），否则会被忽略
3. **数值类型**: 数值列应为数字格式，文本格式可能导致导入失败
4. **覆盖规则**: 相同地区+指标+年份的数据会被覆盖更新
5. **必填列**: region_code 和 region_name 为必填列

## 六、Excel文件制作建议

如果需要自行制作导入文件，可按以下步骤：

1. 新建 Excel 文件
2. 第一行设置为表头：region_code, region_name, E01, E02, ...
3. 从第二行开始填入数据
4. 保存为 .xlsx 格式
5. 通过前端或API导入
