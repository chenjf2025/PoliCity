## 三、Excel批量导入使用方法

### 3.1 Excel格式要求

```
| region_code | region_name | E01 | E02 | E03 | C01 | C02 | ... |
|-------------|-------------|-----|-----|-----|-----|-----|-----|
| default     | 测试城市    | 12.5| 1.85| 58.5| 2.85| 15.6| ... |
| BJ          | 北京市     | 18.5| 3.25| 65.2| 4.20| 22.5| ... |
```

**要求**:
- 第一列: `region_code` (地区代码)
- 第二列: `region_name` (地区名称)
- 第三列起: 各指标编码 (E01, E02, C01, C02, ...)

### 3.2 API导入接口

```bash
curl -X POST "http://192.168.71.127:8000/api/v1/data/raw/import?report_year=2024" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@测试数据.xlsx"
```

### 3.3 单条数据录入API

```bash
curl -X POST "http://192.168.71.127:8000/api/v1/data/raw" \
  -H "Content-Type: application/json" \
  -d '{
    "region_code": "default",
    "region_name": "测试城市",
    "indicator_code": "E01",
    "report_year": 2024,
    "raw_value": 12.50
  }'
```

## 四、触发标准化计算

### 4.1 API调用

```bash
curl -X POST "http://192.168.71.127:8000/api/v1/data/normalize" \
  -H "Content-Type: application/json" \
  -d '{"report_year": 2024}'
```

### 4.2 标准化算法

**正向指标** (polarity=1):
```
P = (x - min) / (max - min) * 100
```

**负向指标** (polarity=-1):
```
P = (max - x) / (max - min) * 100
```
