# CGDSS 城策系统功能总结

## 一、用户认证与权限管理

| 功能 | 说明 |
|------|------|
| **用户注册** | 输入用户名、手机号、邮箱、密码注册，默认待审批状态 |
| **用户登录** | 用户名密码登录，支持7天免登录 |
| **首次登录改密码** | 首次登录强制要求修改密码（需包含大小写字母和数字） |
| **修改密码** | 验证原密码后修改 |
| **忘记密码** | 通过手机号+验证码重置密码 |
| **个人中心** | 查看/修改个人信息（手机、邮箱、姓名） |

### 管理员功能
| 功能 | 说明 |
|------|------|
| **用户审批** | 审批通过/拒绝注册申请 |
| **用户管理** | 创建、编辑、禁用、删除用户 |
| **角色管理** | 普通用户/管理员角色切换 |
| **操作日志** | 查看所有用户的操作记录 |

---

## 二、驾驶舱（首页仪表盘）

| 功能 | 说明 |
|------|------|
| **城市概览** | 展示城市基本信息卡片 |
| **指标雷达图** | 五维评价指标雷达图可视化 |
| **趋势分析** | 核心指标年度变化趋势 |
| **短板提示** | 落后于基准城市的指标提醒 |
| **快捷入口** | 快速跳转到各功能模块 |

---

## 三、指标管理

| 功能 | 说明 |
|------|------|
| **指标列表** | 查看所有评价指标，支持按维度/状态筛选 |
| **指标维度** | 经济活力、产业发展、绿色生态、民生幸福、社会治理 |
| **指标详情** | 查看指标编码、名称、权重、数据来源 |
| **指标维护** | 管理员可新增/编辑/删除指标 |

---

## 四、评价详情

| 功能 | 说明 |
|------|------|
| **雷达图分析** | 多城市多维度对比雷达图 |
| **综合评分** | 城市综合发展水平评分（0-100） |
| **维度详情** | 各维度（经济/产业/生态/民生/治理）得分分析 |
| **趋势分析** | 城市年度评分变化趋势 |
| **短板分析** | 与基准城市对比，找出落后指标 |
| **自定义对标** | 可设置对标城市和阈值 |

---

## 五、政策仿真

| 功能 | 说明 |
|------|------|
| **What-If仿真** | 调整指标数值，预测对综合评分的影响 |
| **历史仿真记录** | 查看历史仿真方案 |
| **AI智能分析** | 基于政策变化，AI分析可能带来的影响 |
| **流式响应** | AI分析结果实时流式返回 |

---

## 六、对标分析

| 功能 | 说明 |
|------|------|
| **城市库管理** | 添加/管理对标城市 |
| **对标城市选择** | 按城市级别（一线/二线/三线）筛选 |
| **多城市对比** | 最多5个城市同时对比 |
| **维度对比** | 各维度得分对比分析 |

---

## 七、AI助手

| 功能 | 说明 |
|------|------|
| **智能问答** | 基于Dify的AI对话，解答系统使用问题 |
| **会话管理** | 多会话切换，持久化存储 |
| **对话复制** | 一键复制AI回答内容 |
| **收藏功能** | 收藏重要对话 |
| **导出对话** | 导出会话为Markdown文件 |
| **追问建议** | AI回答后推荐可追问的问题 |

---

## 八、系统架构

```
前端 (Vue3 + Element Plus + ECharts)
    ↓ HTTP/API
后端 (FastAPI + SQLAlchemy)
    ↓
数据库 (PostgreSQL)
    ↓
Redis (缓存)
```

**技术栈**：
- 前端：Vue3 + TypeScript + Vite + Vue Router + Pinia
- 后端：Python FastAPI + SQLAlchemy + Pydantic
- 数据库：PostgreSQL + Redis
- 部署：Docker + Docker Compose + Nginx
